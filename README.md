# vue3-project（vue3+ts实战项目）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

+ 支持搜索会议名字
+ 会议内容
+ 时间作为时间轴
+ 会议名字，一个会议对应多个预定记录
+ 过期时间禁用不能预约
+ 主要是鼠标点击的时候、鼠标不间断的移动、鼠标抬起、鼠标移动、鼠标移出
